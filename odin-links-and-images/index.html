Links

When creating links we have two options:

  • Absolute Links
  • Relative Links

Absolute Links

An Absolute Link is a link to a page on another website.

Relative Links

A Relative Link links to pages on the same website.

They do not include the domain (Eg Http://www.websitename.com).

Relative links only include the file name relative to the current page

For example, if we have a page (about.html) in the same directory, we can link to it by using just 'about.html' in the href, Like this

When looking for files in directories it is recommended to include `./` before the link.

If you would like to look in a parent directory you can use `../`
